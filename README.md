### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/huddle-landing-page.git

  cd huddle-landing-page

  pnpm install
```

- for running dev server

```bash
pnpm run dev
```

- for build & review

```bash
pnpm run build
pnpm run preview
```
